# 📌 Task Manager (MERN + MySQL)

A simple Task Management Web Application where users can register, login, and manage their tasks efficiently.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 📋 Task Management (Create, Read, Update, Delete)
- 📊 Dashboard with task statistics
- 🎨 Modern and responsive UI
- ⚡ Loading indicators during API calls
- ❗ Error (red) and success (green) alerts
- ✅ Form validation

---

## 🛠️ Technologies Used

### Frontend
- React.js
- CSS (Custom UI)
- Axios

### Backend
- Node.js
- Express.js
- JWT

### Database
- MySQL

---

## 📱 UI/UX Improvements

- Loading spinner during API calls
- Disabled buttons while loading
- Error messages in red alert boxes
- Success messages in green alert boxes
- Responsive design (mobile, tablet, desktop)
- Smooth hover and transition effects

This project demonstrates:

- Authentication system
